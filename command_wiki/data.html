・block
 <if | unless> < block> <x y z>

指定した座標のブロックとブロックが一致している(unlessの場合一致していない)時にコマンドを実行します。


・blocks
 <if | unless> < blocks> <x y z(始点)> <x y z(終点)> <x y z(比較先)> <all | masked>

　　・all
　　　　範囲内のすべてのブロックを比較する。
　　・masked
　　　　範囲内の空気ブロックは無視してひかくする。


・data[ Java Edition限定? ](ver1.14で追加)
 <if | unless> data <block | entity | storage> <データタグの種類>

指定したブロック、エンティティ、ストレージ(ver1.15以降)が指定したデータタグを持っている(unlessの場合持っていない)時にコマンドを実行します。

　　・entity
　　　　
 entity <セレクタ>

　　　　エンティティを指定します。
　　・block
　　　　
 block <x> <y> <z>

　　　　ブロックエンティティを指定します。
　　・storage(ver1.15で追加)
　　　　
 storage <ストレージ名>

　　　　ストレージ(タグを一時的に保存しておく場所)を指定します。ストレージ名は任意の文字列が使えます。

データタグの種類[ 展開 ]

データタグの種類
AでタグAを選択します。
{}でデータタグ全体を選択します。
{A:x}でタグA:xが含まれている場合に限りデータタグ全体を選択します。
A{}でCompound型のタグAを選択します。
A{B:x}でタグB:xを含むCompound型のタグAを選択します。
A[]でList型のタグAのすべての要素を選択します。
A[x]でList型のタグAの(x+1)番目の要素を選択します。
A[{}]でList型のタグAに含まれる{}で区切られた要素をすべて選択します。
A[{B:x}]でList型のタグAに含まれる{}で区切られた要素のうちタグB:xを含むものを選択します。
A.BでCompound型のタグAに含まれるタグBを選択します。
