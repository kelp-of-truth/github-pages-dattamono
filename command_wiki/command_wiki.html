execute

executeコマンド

  /execute <サブコマンド> [サブコマンド] ... run <実行コマンド>

サブコマンド概要
as
 as <セレクタ>

指定したエンティティとして実行
実行者->変わる(as)
向き->変わる
実行位置->変わらない
positioned
 positioned <x> <y> <z>

指定した座標で実行
実行者->変わらない
向き->変わらない
実行位置->変わる(positioned)


 positioned as <セレクタ>
実行者->変わらない
向き->変わらない
実行位置->変わる(positioned)
rotated
 rotated <y軸回転(横)> <x軸回転(縦)>

指定した角度として実行
実行者->変わらない
向き->変わる(rotated)
実行位置->変わらない


 rotated as <セレクタ>

指定したエンティティと同じ角度として実行
実行者->変わらない
向き->変わる(rotated)
実行位置->変わらない
facing
 facing <x> <y> <z>

指定した座標を向いているとして実行
※向いているとして実行しているだけなので見た目は変わらない
実行者->変わらない
向き->変わる(facing)
実行位置->変わらない


 facing entity <エンティティ> <feet | eyes>

feetの場合、足、eyesの場合、<エンティティ>で指定したエンティティの目向いているとして実行
実行者->変わらない
向き->変わる
実行位置->変わらない
in
 in <overworld | the_nether | the_end>

指定したディメンションで実行
実行者->変わらない
向き->変わらない
実行位置->変わる(in 別ディメンションで)
at
 at <セレクタ>

指定したエンティティの場所と向きで実行
実行者->変わらない
向き->変わる(at)
実行位置->変わる(at)
anchored
 anchored <feet | eyes>

実行者の基準の場所を変える
※実行位置が変わるわけではなく、ローカル座標やfacingにのみ影響する
デフォルトではfeetです。
[ Java Edition限定? ]
align
 align <x | y | z | xy | yz | xz | xyz>

指定した軸の座標を整数にする(1.13は座標が型になったため)
これは小数点以下を切り捨てた値です
if unless
 if <block | blocks | data | entity | predicate | score>
